Orbit Sim

A Domain Specific Language for Rocket Flight Simulation

About

Orbit Interpreter is a domain-specific language designed to simulate rocket flights with simple, readable syntax.

View on GitHub

Features

  • Define custom rockets with physical properties
  • Simulate different gravitational environments (Earth, Mars, etc.)
  • Create complex flight plans with time and altitude triggers
  • Visualize rocket trajectories in terminal
  • Generate altitude and range reports

Example Code


// Simple vertical rocket launch demonstration
rocket SmallRocket {
  mass = 1.0 // kg (total mass)
  fuel = 0.7 // kg (fuel mass)
  thrust = 25 // N (otal thrust)
  burnRate = 0.1 // kg/s (fuel consumption)
  diameter = 0.05 // m (diameter of the rocket)
  Cd = 1.14 // drag coefficient (assumed for a simple rocket)
}
flight Vertical {
  at t=0 {
    angle = 85 // almost vertical
    power = 100% // full throttle
  }
}

simulate Vertical {
  display trajectory
  report max_altitude
  report range

}
}

Getting Started

Run your orbit simulation with a simple command:

python orbit_interpreter.py yourfile.orb
