Orbit Interpreter

A Domain-Specific Language for Rocket Flight Simulation

Simulate Rocket Flights with Simple Code

Orbit Interpreter is a powerful yet easy-to-use language designed specifically for simulating rocket flights. Define rockets, environments, and flight plans with a simple syntax, then watch your rocket soar through simulated space!

Key Features

Rocket Definition

Define rocket properties like mass, fuel, thrust, burn rate, and drag coefficient.

Environment Simulation

Simulate different gravitational and atmospheric conditions like Earth or Mars.

Flight Plans

Create complex flight plans with time- and altitude-based events.

Trajectory Visualization

See your rocket's path with text-based trajectory visualization.

Physics Simulation

Accurate physics modeling including gravity, thrust, and atmospheric drag.

FizzBuzz Integration

Fun FizzBuzz functionality for deployment notifications.

How It Works

Writing code in Orbit is simple and intuitive. Here's an example:

// Define your rocket
rocket MyRocket {
  mass = 50.0
  fuel = 20.0
  thrust = 1000
  burnRate = 0.5
  diameter = 0.3
  Cd = 0.5
}

// Define the environment
environment Earth {
  gravity = 9.81
}

// Create a flight plan
flight TestFlight {
  at t=0 {
    angle = 90
    power = 100%
  }
  at t=10 {
    angle = 45
    power = 50%
  }
  at altitude=1000 {
    angle = 20
    power = 30%
  }
}

// Run the simulation
simulate TestFlight {
  display trajectory
  report max_altitude, range
}

Just save this as a .orb file and run it with:

python orbit_interpreter.py your_file.orb

Getting Started

Follow these simple steps to start creating your own rocket simulations:

  1. Clone the repository: 
    git clone https://github.com/yourusername/orbit-interpreter.git
cd orbit-interpreter
  2. Create your first .orb file
  3. Run the interpreter: 
    python orbit_interpreter.py your_file.orb

No external dependencies required! Just Python 3.8 or higher.
